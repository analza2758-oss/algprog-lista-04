## 📌 Descrição

Os exercícios têm como objetivo praticar conceitos fundamentais de programação, como:

* Estruturas de repetição (`for`, `while`)
* Estruturas condicionais (`if`, `else`)
* Entrada de dados com `Scanner`
* Validação de dados
* Operações matemáticas
